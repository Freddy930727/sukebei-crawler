介紹:
這個程式可以幫你判斷該torrent是不是你指定的那天的，且按照你喜好的種類去下載，我爬的這網站應該是我所知最多人在用且分類最整齊、廣告最少的網站(piratebay除外)，所以不用擔心速度會不會太慢。

注意事項:
1.只有64位元的電腦才能執行，我日後會處理(mac我不確定，因為我是用win10)
3.chrome要更新到最新版本
4.家裡有色情守門員的人有兩種選擇，一個是電腦直接掛vpn，另外一個是用cus_main.exe ，這裡面會自動幫你連vpn但問題是，那vpn的速度很不穩定，有很高機率會失敗或斷線，所以建議用第一個方法
5.會被防毒軟體刪除，解決方法就是把防毒軟體暫時關掉，並把被他刪掉的檔案還原或重新下載
6.下載好的位置如同你chrome下載預設的位置
7.github:，有興趣的可以參考
8.有問題的話可以email我(930727fre@gmail.com)，如果你跟我在同個群組，也可以直接私訊我
